## 📝 Documentation Conventions

### Code Examples

Code examples are provided in the following format:

```javascript
// JavaScript/Node.js examples
const example = "like this";
```

```bash
# Shell commands
npm install
```

### API Endpoints

API endpoints are documented as:

```
METHOD /api/endpoint
```

Example:
```
GET /api/home/public/en/hero
POST /api/auth/login
```

### Notes and Warnings

Important information is highlighted:

**Note**: Additional information or tips

**⚠️ Warning**: Important warnings about potential issues

**✅ Best Practice**: Recommended approaches
